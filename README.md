# Swift Video Convert

Fast, lightweight macOS video converter with smart codec detection and remuxing.

## Features

- **Smart Remuxing**: Automatically copies compatible codecs (30 seconds vs 30 minutes)
- **Format Support**: MP4, MOV, MKV, WebM, AVI
- **Manual Control**: Override codecs, quality, speed when needed
- **Native macOS**: Built with SwiftUI

## Requirements

- macOS 13.0+
- ffmpeg (`brew install ffmpeg`)

## Installation

1. Clone the repo
2. Open `swiftconvert.xcodeproj` in Xcode
3. Build and run
